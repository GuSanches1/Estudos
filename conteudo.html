Branch

Primeiro acesse o ponto que voce quer fazer uma branch com o checkout + id do commit, dps de git checkout -b que significa que vc quer criar uma branch e logo apos de o nome da branch

Para adicionar o conteudo da branch no arquivo, basta utilizar o comando merge, mas para utilizar ele tem q estar na master

Aula 3, colaboração com github

primeiro digite - (git remote add origin https://github.com/GuSanches1/CursoJS.git) para determinar o seu repositorio no github

depois renomie a master, se desejar com o comando git branch -M main

e de um git push -u origin main para adicionar os conteudos


Depois de atualizar o o codigo, mande um git add . para adicionar, dps o commit e de um git push para enviar para github

Para retirar o arquivo do github para a sua maquina, vc deve determinar um caminho utilizando cd + o caminho desejado, dps de um git clone + o link do arquivo

para fazer a atualizaçao de um cod de empresa por exemplo, vc vai pegar o fork no github, adicionar na sua maquina com o passo a passo logo acima e dps criar uma branch para começar as alterações

para crair uma branch no git, tem q dar um git push + nome da branch e no github abrir um request para adicionar no cod novo
